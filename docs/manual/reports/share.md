# 리포트 공유 

## PDF 다운로드 
생성된 리포트는 출력을 위해 PDF 파일로 변환하여 다운로드할 수 있어요.

![이미지](./img/example.png)
<p></p>

PDF 파일 변환 후 다운로드를 클릭하면 학생 리포트가 다운로드 돼요.

## 모바일 발송 <Badge type="tip" text="대표교사" />
리포트는 모바일 알림톡으로도 발송할 수 있습니다.

![이미지](./img/example.png)
<p></p>

엑셀파일을 다운받아 리포트를 발송할 연락처를 입력 후 업로드해 주세요.

![이미지](./img/example.png)
<p></p>
